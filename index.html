Por que Elixir?

Elixir é uma linguagem de programação dinâmica e funcional, usada para construir aplicações escaláveis desenvolvida pelo brasileiro José Valim em 2012.

Scalability

All Elixir code runs inside lightweight threads of execution (called processes) that are isolated and exchange information via messages.

Fault-tolerance

The unavoidable truth about software running in production is that things will go wrong. Even more when we take network, file systems, and more.

Extensibility

Elixir has been designed to be extensible, letting developers naturally extend the language to particular domains, in order to increase their productivity.
